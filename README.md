# Uniqon Garage Management System - Technical Documentation

### Developed by: George Eipa
**Sales Manager at EMCA MOTORS | Front-end Developer**

## 📌 Project Overview
This project is a comprehensive technical documentation page for the **Uniqon System** used at EMCA MOTORS garage. It serves as a central guide for managing sales, inventory, and garage operations while demonstrating responsive web design principles.

## 🛠 Technologies Used
* **HTML5:** Semantic structure for technical content.
* **CSS3:** Custom styling including a fixed navigation sidebar and code block formatting.
* **Media Queries:** Fully responsive design optimized for mobile, tablet, and desktop views.

## 🚀 Key Features
* **Interactive Navigation:** A fixed sidebar for quick access to different system modules.
* **Mobile-First Approach:** Ensures the garage team can access documentation on the go via smartphones.
* **Code Samples:** Technical snippets for system configuration and management.
